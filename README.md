# LUNES - Time to be You

![LUNES](https://img.shields.io/badge/Status-Production-green)
![React](https://img.shields.io/badge/React-18.3-blue)
![TypeScript](https://img.shields.io/badge/TypeScript-5.6-blue)
![Vite](https://img.shields.io/badge/Vite-6.0-purple)

LUNES é uma plataforma de experiências holísticas que conecta pessoas com o seu tempo e essência através de quatro pilares fundamentais: **MOVE**, **EXPLORE**, **FEEL** e **STAY**.

## 🌟 Sobre o Projeto

**"Time to be You"** - LUNES não vende tempo. LUNES devolve o tempo.

Este projeto representa uma identidade visual e digital única, criada por Luísa e Francisco, fundindo os seus apelidos (Luz + Nunes) numa experiência que convida cada pessoa a viver ao seu próprio ritmo.

## 🎯 Pilares do Ecossistema

### 🏃 MOVE - Movimento com Alma
Treino funcional personalizado num espaço boutique que fortalece corpo e comunidade.

### 🚤 EXPLORE - A Liberdade do Horizonte
Experiências náuticas curadas que conectam com a natureza selvagem do Algarve.

### 🌿 FEEL - A Raiz e a Nutrição
Agricultura biológica certificada e experiências farm-to-table autênticas.

### 🏡 STAY - O Refúgio do Ser
Alojamento minimalista desenhado para "estar", sem a urgência de "fazer".

## 🚀 Tecnologias

- **React 18.3** - Framework principal
- **TypeScript 5.6** - Tipagem estática
- **Vite 6.0** - Build tool & dev server
- **Framer Motion** - Animações fluidas e interativas
- **Tailwind CSS** - Styling utility-first
- **Lucide React** - Ícones modernos
- **React Router** - Navegação SPA

## 📦 Instalação

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/lunes.git

# Entre no diretório
cd lunes

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

## 🛠️ Scripts Disponíveis

```bash
# Desenvolvimento
npm run dev          # Inicia o servidor de desenvolvimento

# Produção
npm run build        # Cria build de produção
npm run preview      # Preview do build de produção

# Qualidade de Código
npm run lint         # Executa ESLint
```
